# UKA Valgomat (Vite + React + TS + Tailwind)

## Quick start
```bash
cd uka-valgomat
npm install
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173).

## Build
```bash
npm run build
npm run preview
```

## Notes
- Requires Node.js 18+.
- All role logic is in `src/UKAValgomat.tsx` (arrays `QUESTIONS` and `POSITIONS`).
